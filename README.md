# UI-UX-Experiment4

## Aim:

To design a wireframe for the “CityPulse” mobile application using Balsamiq with essential screens and proper user navigation.

## Algorithm:

Step 1 :

Open the Balsamiq Wireframing tool.

Step 2:

Create a new wireframe project.

Step 3:

Add mobile device layouts for each screen.

step 4:

Design the Splash Screen with logo and buttons.

step 5:

Create the Home Dashboard with categories and navigation.

step 6:

Design the Submit Complaint screen with input fields.

step 7:
Create the Track Complaint screen with status tracking.

step 8:

Design the Notifications screen with alert cards.

step 9:

Create the Profile & History screen with user details.

step 10:

Add navigation flow between all screens.

step 11:

Align and organize all UI components properly.

step 12:

Export the completed wireframe design.
